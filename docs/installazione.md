# Installazione

## Prerequisiti:

- Per installare il progetto è necessario aver installato GIT per clonare il repository
    bash**
    git --version   ** per verificare che sia installato **


- Un browser moderno (Chrome, Firefox, Safari)
- Visual Studio Code (consigliato) per modificare il progetto

## Clonare il repository

bash#

git clone https://github.com/tuo-utente/tuo-progetto   **esempio per clonare il repository**

- Entra nella cartella:

bash#

cd tuo-progetto

## Avvio del progetto

- Aprire la cartella del progetto in Visual Studio Code
- Fare clic destro su index.html
- Selezionare "Open with Live Server"

## Aggiornare il progetto

    bash**
    git pull origin main
